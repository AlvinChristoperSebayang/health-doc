---HealthDoc HTML Template---

Welcome to the HealthDoc HTML Template! This template is designed to help you create a modern and professional online presence for your healthcare application or medical service. With its clean design and responsive layout, it ensures an optimal viewing experience across all devices.

Features
Responsive Design: Ensures seamless display on desktops, tablets, and mobile devices.
Smooth Animation Transitions: Enhances user engagement with fluid and visually appealing animations.
Comprehensive Healthcare Components: Includes patient registration forms, appointment scheduling, doctor dashboards, and medical history tracking.
Intuitive Navigation: Simplifies access to various sections of the application for both patients and healthcare professionals.
SEO Friendly: Optimized code structure for better search engine visibility.
Getting Started
Follow these instructions to set up the project on your local machine:

Extract the zip files.
Open in your code editor.
Open terminal and type npm i (Make sure you have Node.js installed on your device).
Use npm run dev to run development mode.
